Prime.vc

Data driven GUI application libraries, for human beings and Haxe.
Take a look at the sample project with introduction to the framework concepts.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/touch/PrimeVC
