currentTime(null,null) : prime.bindable.Bindable<Float>

Number of seconds the stream has been playing. This value is automaticly updated, as long as the stream is playing. The first time the time is called, the stream will start a timer to update the value (every 200ms). Note: To seek the in the stream, you should not change this property but use the "seek" method.

Current state of the media-stream. Do not modify the state directly, but use the appropriate methods to change the state of mediastream.
