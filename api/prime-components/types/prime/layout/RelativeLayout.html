The relative layout class describes the wanted settings of a layout client in relation with the layout-parent. In a relative layout it's possible to set the prop horizontalCenter = 0 to make sure the layout is always centered in relation with its layout parent. The parent can choose to apply or ignore the relative properties depending on the layout-algorithm that is uses. When the algorithm is positionating every layout-child next to eachother, the horizontalCenter will be ignored for example. This class is not meant for standalone use but should always be placed in the LayoutClient.relative property.
