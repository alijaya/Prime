A group of Signal instances and (when applicable) other signal groups. Signals allows to pool Signal instances in one place. It implements the haxe.Public interface to make signals public by default.

Usage example of a group with 1 Signal and another signal group

class LoaderSignals extends Signals { var unloaded (default, null) : Signal0; var load (default, null) : CommunicationEvents; }
