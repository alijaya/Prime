inline polygonRadiusForCircle(radius : Float, sides : Int) : Float

parameters
radius radius of the circle to cover
sides sides of the polygon
returns
radius of the polygon to cover the circle
Method will return the radius of a polygon that is needed to cover a circle. This is useful if you want to draw a part of a masked polygon to fill a piece of circle. Using the radius of the circle would leave a part of the circle uncovered.
