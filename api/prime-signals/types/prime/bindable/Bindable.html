Class to keep a value automatically updated. You can trigger another bindable to update by doing: var a = new Bindable (5); var b = new Bindable (6); a.bind(b); //a will be 6 now b.value = 8; //a will be 8 now You can also create a two way binding by doing: a.pair(b); Which is effictively the same as doing: a.bind(b); b.bind(a); //will not create an infinte loop ;-) You can trigger a method when the property is changed: using prime.utils.Bind; function updateLabel (newLabel:String) : Void { textField.text = newLabel; } var a = new Bindable ("aap"); updateLabel.on( a.change, this ); a.value = "2 apen"; //textField.text will also be changed now The 'change' event will be dispatched after 'this.value' changes.
