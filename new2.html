Introduction

Prime is a collection of 12 open-source Haxe libraries that are designed to make building cross-platform, data-driven applications more productive and fun. Each library is focussing on their own problem-solution.

The libraries are designed to be pluggable, meaning that using one library does not mean you have to use the other libraries.

Installation

To install all libraries and the scaffolding tool run:

haxelib install prime

Or install each library seperatly by running:

haxelib install prime-{lib-name}

Libraries

  • Core

    Core interfaces and generic utilities.

    Stable All

  • Signals

    Strictly typed, lightweight, memory and cpu optimized signal (event) library.

    Stable All

  • Bindable

    Data-binding built on signals.

    Stable All

  • i18n

    Internalization library. Compiles YAML into bindable output.

    Stable All

  • MVC

    Strictly typed Model-View-Controller for stateful GUI applications.

    Stable All

  • FSM

    Finite State Machine, with state change events built on signals.

    Stable All

  • Layout

    2D box-model layouting for anything.

    Stable All

  • Display

    Flash Display-list as a datastructure with signals and change events.

    Stable SWF

  • Components

    GUI components seperated into: logic, skinning and styling.

    Stable SWF

  • Media

    Video and Audio player stream state handling and components.

    Stable SWF

  • Data

    Generic data handling. ValueObjects, CSV parser and other utils.

    Alpha All

  • Perceptor

    Firebug-like inspector.

    Alpha SWF
